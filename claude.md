# 멍멍 스테이 프로젝트

## 프로젝트 개요

프리미엄 애견 펜션 "멍멍 스테이"의 랜딩 페이지 웹사이트입니다.

## 기술 스택

- HTML5
- CSS3 (styles.css)
- Vanilla JavaScript (script.js)
- Google Fonts (Noto Sans KR, Outfit)
- FontAwesome Icons

## 파일 구조

```
webbill/
├── index.html      # 메인 HTML 파일
├── styles.css      # 스타일시트
├── script.js       # JavaScript 파일
├── about.png       # 소개 섹션 이미지
├── STRUCTURE.md    # 웹사이트 구조 문서
└── claude.md       # 프로젝트 가이드
```

## 주요 섹션

| 섹션 ID | 설명 |
|---------|------|
| `#navbar` | 고정 네비게이션 바 |
| `#home` | 히어로 섹션 |
| `#about` | 펜션 소개 |
| `#amenities` | 시설 안내 (수영장, 운동장, 스파) |
| `#rooms` | 객실 안내 (Suite A, Suite B) |
| `#gallery-cta` | 예약 유도 CTA |
| `footer` | 푸터 (연락처, SNS) |

## 개발 가이드라인

- 한국어 콘텐츠 기반 (`lang="ko"`)
- 반응형 디자인 적용 (모바일 버거 메뉴)
- 스크롤 애니메이션 사용 (`data-scroll` 속성)
- 이미지는 lazy loading 적용

## 브랜드 정보

- **브랜드명**: 멍멍 스테이 (MungMung Stay)
- **컨셉**: 반려견과 함께하는 프리미엄 휴식 공간
- **위치**: 강원도 속초시
